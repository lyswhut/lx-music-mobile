<p>洛雪音乐是一款音乐查找和播放工具，基于 React native 开发。</p>
<h3>自由开源</h3>
<p>落雪音乐是一款在 Apache 2.0 协议下授权和发布的<strong><a href="https://www.gnu.org/philosophy/free-sw.html">自由软件</a></strong>。本项目开源发布于 GitHub。你可以自由地运行，拷贝，分发，学习，修改并改进落雪音乐。</p>
<h3>功能丰富</h3>
<p>落雪音乐支持多种格式和在线音乐源。您可以很方便地检索和导入它们。落雪音乐支持对音乐源订阅并即使获取更新。</p>
<p>落雪音乐提供了访问互联网歌单、歌曲列表备份、音乐缓存、音乐下载、不喜欢歌曲规则歌词、翻译桌面歌词等丰富的功能，拥有强大的可定制性。</p>
<h3>优秀的跨平台和多端互通特性</h3>
<p>本项目支持在Windows、Mac OS、GNU/Linux 以及 Android 平台上运行。导出的歌曲列表在三个平台上均可使用。</p>
<h3>非凡的健壮性</h3>
<p>落雪音乐项目起始于 2021 年，拥有三年历史。项目迄今为止已接受 520 个以上的 Github Issues 和 20 个以上的 Pull Requests ，解决绝大多数的已知漏洞。在时间的淬炼中获得了卓越的稳定性。</p>
<h3>友好的社区</h3>
落雪音乐有官方 QQ 群，Github Issues 长期活跃。
<h3>注意事项：</h3>
<p>本项目不计划支持 IOS。当前版本比 128k 更高的音质已失效。由于收到腾讯投诉要求停止提供软件内置的连接到他们平台的在线播放及下载服务，所以从2023年10月18日起不再提供此服务。你不应该在任何公共场合对落雪音乐做任何宣传。落雪音乐一直为免费使用的软件，如果你是从某些地方买的话，那末你就是被骗了。QQ群仅用于反馈。我们推荐使用 GitHub 反馈问题，只有在你不了解 GitHub 的情况下，才可以在QQ群提供建议或反馈。在反馈前，阅读并确保你你知悉<a href="https://docs.lxmusic.top/#%E5%A6%82%E4%BD%95%E5%8F%8D%E9%A6%88">落雪音乐官网上的《如何反馈》提示。</a>请在提出 Github Issues 后提出 Pull Request。</p>
